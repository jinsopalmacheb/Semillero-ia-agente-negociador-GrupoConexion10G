## Descripción
Agente inteligente que detecta la intención de pago del cliente y la fecha de compromiso para registrar una promesa de pago.

## Tecnologías
- Python
- OpenAI (conceptual)
- GitHub

## Funcionamiento
El agente inicia una conversación, analiza el mensaje del cliente, identifica intención y fecha, y genera una promesa de pago.
